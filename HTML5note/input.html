input控件


1.input标签为单标签，type属性为其最基本的属性，其取值有多种，用于指定不同的控件类型（type：控件）


2.input标签还可以定义很多其他的属性

属性 属性值 描述
type text 单行文本输入框
password（密码） 密码输入框
radio 单选按钮
checkbox 复选框
button 普通按钮
submit 提交按钮
reset 重置按钮
image 图像形式的提交按钮
file 文件域
name 由用户自定义 控件的名称（用于分类选中项）
value 由用户自定义 input控件中的默认文本值
size 正整数 input控件在页面中显示的宽度
checked checked 定义选择控件默认被选中的项
maxlength 正整数 控件允许输入的最多字符数

3.表示默认选择状态：checked = "checked"


4.只有图像按钮才能这样写：<input type = "image" src = "">
