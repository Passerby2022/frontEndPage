label标签


1.label标签为input标签的元素定义标注


2.作用：用于绑定一个表单元素，当点击label标签的时候，被绑定的表单元素就会获得输入焦点


3.for属性规定labe与哪个表单元素绑定

<label for = "male">Male</label>

<input type = "radio" naem = "sex" id = "male" value = "male">


4.点框以外的地方也能将鼠标的输入定位在框中（用label之间进行包裹）

<label>用户输入：<input type = "text"></label>


5.如果label里面有多个表单，想要定位到某一个，可以通过for id的格式来进行（与锚点定位相似）
